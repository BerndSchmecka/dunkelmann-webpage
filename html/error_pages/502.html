502 Bad Gateway

Du bekommst diese Meldung angezeigt, da zwar der Hauptserver für dunkelmann.eu einwandfrei funktioniert, aber der dahinterliegende Server für %host% nicht zu erreichen ist. Dies kann mehrere Ursachen haben, vermutlich ist der Server abgestürzt oder wird kurz neugestartet. Für längere, geplante Wartungen wird in der Regel eine Seite mit Statuscode 503 vorgeschaltet, die über die Wartungen informiert.
